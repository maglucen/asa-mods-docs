# Installation

## Client installation

Subscribe to the mod from its ARK: Survival Ascended mod page and let the game download it before joining a server that uses it.

## Server installation

Add the mod to your server mod list using the mod ID shown on the mod page.

Restart the server after changing the mod list or editing configuration values.

## Recommended update flow

1. Stop the server.
2. Update the mod.
3. Review the changelog for config changes.
4. Update `GameUserSettings.ini` if needed.
5. Start the server.
6. Check the server logs if the expected behavior does not appear in game.
