# Toggle Loot Quality Buffs Configuration

Add the configuration section for this mod to `GameUserSettings.ini`.

```ini
[ToggleLootQualityBuffs]
Enabled=true
```

## Settings

### `Enabled`

Controls whether the loot quality buff behavior is active.

- `true` = enabled.
- `false` = disabled.
